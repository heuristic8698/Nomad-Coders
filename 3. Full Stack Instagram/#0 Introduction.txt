#0-1 What are we using
visual studio code
 - 혹은 기타 텍스트에디터(Atom, Bracket, Sublime)
 - visual studio code는 integrated terminal 기능으로 command 바로 실행 가능
xcode or android studio
 - 맥, iOS 개발은 xcode
 - 안드로이드 개발은 android studio
genymotion
 - android studio 설치 없이 테스트를 하고 싶으면 genymotion 필요
expo, nodejs, npm, yarn 
 - 관련 내용은 reactjs web 코딩과 react native 날씨앱 코딩 참조
 - nodejs, npm, yarn는 프런트엔드와 앱을 위해서 필요
Python3, pip, pipenv
 - Python은 다른 버전이 아니라 3을 설치해서 터미널에서 실행할 수 있어야 함
 - Python3, pip, pipenv 모두 벡엔드를 위해 사용
postgres
 - 맥 유저는 postgres app, 윈도우 유저는 pg admin
expo client
 - device에서 직접 테스트할 때 사용
 
#0-2 Requirements
 - React JS, React Native 무료 강의 수강
 - 기초적인 Python, JSP 문법, variable, function, class개념

#0-3 Check if everything works
python, pip 검사
 - VSC terminal에서 python 입력, python 2.x 버전이 설치되어 있었다면 python3 으로 확인
 - pip은 3.4 버전 이상을 공식 사이트에서 다운 받으면 자동으로 설치
 - 커맨드에서 pip을 입력하면 도움말이 뜨면 설치 확인
pipenv 설치 및 확인
 - pipenv는 링크에 문제가 있어서 커맨드 창에서 'python -m pip install --user pipenv' 입력해서 설치
 - 설치 한 다음에 나타난 메시지 참고해서 환경변수 Path에 경로 추가
 - 새 cmd 창에 pipenv를 입력해서 설치 확인
nodejs, npm, yarn 검사
 - cmd에 'node -v', 'npm', 'yarn'
